Git

  • git init - создание нового репозитория
  • git clone [ссылка] - копирование удаленного репозитория
  • git status - проверка состояния файлов
  • git add [файл] - добавление файла в индекс
  • git add . - добавление всех изменений
  • git commit -m "сообщение" - сохранение изменений
  • git push - отправка на GitHub
  • git pull - получение изменений с GitHub
  • git branch - список веток
  • git checkout [ветка] - переключение на ветку
  • git merge [ветка] - слияние веток
